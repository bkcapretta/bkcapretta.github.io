Music Cognition Research

Fall 2017

Last Fall, I researched with Music Psychologist Aniruddh Patel if a musical synchronous activity between two people leads to an increase in intentional binding. I studied background research on joint music making, group cohesion, sense of agency, and self-other blurring. Ultimately, I coded three separate experiments in Javascript and HTML/CSS to track intentional binding on EvoThings using Android development tools.

Work for this research can be found here.

Machine Structure and Assembly Language

Spring 2017

  • Interfaces, Implementations, and Images: implemented a two-dimensional, polymorphic, unboxed array using Hanson's C interfaces and implementations, and then used that implementation to identify Sudoku solutions
  • Cache and Locality: implemented blocked two-dimensional arrays, and then used it to evaluate the performance of image rotation using three different array-access patterns with different locality properties
  • Integer and Logical Operations: given a PPM image, compressed the image by transforming color spaces and discarding info using lossy compression. There was a lot of packing and unpacking of binary data, work with two's complement, and floating-point arithmetic.

    Here is a funny picture of what the compressed image looked like half-way through the project.

  • Analysis of AMD64 Assembly Code: read, analyzed, and understood a half-dozen or so C procedures given access only to an executable binary. In the process of doing so, learned how to use two debuggers (GDB and DDD)
  • Assembly-Language Programming: implemented a calculator that used Reverse Polish Notation in assembly language
